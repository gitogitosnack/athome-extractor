# [リポジトリ名]

Athomeの物件検索結果ページ（HTMLソース）から、物件情報を一括で抽出するためのスクリプトです。

## 概要
本ツールは、ブラウザのデベロッパーツール（DevTools）から保存したHTMLファイルを読み込み、物件名称、価格、所在地、間取りなどの主要な情報を抽出して[CSV/JSON等]形式で出力します。

## 操作手順

物件情報の抽出は以下のステップで行います。

### Step 1: HTMLソースの取得
1. ブラウザで「Athome」の物件一覧ページを開き、任意の条件で検索します。
2. 目的のページが表示されたら、`F12` キー（または右クリック > 検証）でデベロッパーツールを開きます。
3. `Elements` タブ内の最上部にある `<html>` タグを右クリックし、 **[Copy] > [Copy outerHTML]** を選択します。

### Step 2: ファイルの保存
1. コピーした内容を新しいファイルとして保存します。
2. 保存先は以下のディレクトリに格納してください。
   - **保存場所:** `data/raw_html/`
   - **ファイル名:** `[任意名称].html` （例: `search_results_01.html`）
   ※ 複数のページを解析したい場合は、それぞれ別名で保存して同じフォルダに入れてください。

### Step 3: スクリプトの実行
1. ターミナルでリポジトリのルートディレクトリに移動します。
2. 以下のコマンドを実行して抽出を開始します。

```bash
# 実行コマンドの例（Pythonの場合）
python extract_properties.py
```

### Step 4: 結果の確認
1. 抽出が完了すると、以下の場所に結果ファイルが生成されます。
   - **出力先:** `data/output/`
